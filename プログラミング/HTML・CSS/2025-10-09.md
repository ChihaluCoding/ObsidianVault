
# URLとは？
### リソースの場所を示す
- 正式名称 Uniform Resource Locator
- Uniform：統一された
- Resource：資源
- Locator：場所を示すもの


# HTTP / HTTPS とは？

#### 通信するときに使うプロトコル

**HTTP（HyperText Transfer Protocol）**  
-  ウェブブラウザとサーバーがテキストや画像などをやり取りするときのルール  
-  ただし **暗号化されていない**、通信内容は途中で見られる可能性がある

**HTTPS（HTTP Secure）**  
-  HTTPに **暗号化（SSL/TLS）** が追加されたもの  
-  送受信するデータが暗号化されているから、より安全
